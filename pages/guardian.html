Description

My friend Albert and I reached the conclusion we needed a private file server for sharing our geeky stuff. We could have used cloud solutions like Google drive or Dropbox. But we preferred the approach "Do it by yourself". I built the first prototype in a couple of days and after that I spent a little bit of time brushing the code and make it more cool. The UI is pretty simple and it has a few features. Why nothing else? Because it's all we need.

Guardian is an http microservice writen in Golang. It serves static files from the server. It uses jwt as authentication process to restrict the access of files.

After building Guardian I can say that Golang performance is quite amazing. You can upload 1gb files in 300-400 ms. And all of that using practically only the standard library.
